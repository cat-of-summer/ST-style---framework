Эффекты ST-style — все кейсы

Подключён только ../dist/effects.min.css. Каждый чип управляется атрибутами. Наводи курсор на hover-карточки, кликай по active/ripple, а анимации показаны в infinite для наглядности (рядом — их одноразовый вариант с кнопкой «проиграть»).

Длительность перехода — [td]

Управляет transition-duration элемента и его псевдоэлементов. Значения из карты $durations: rapid 0.1s · fast 0.325s · medium 0.55s · slow 0.775s · 1s…15s · none. Наведи — почувствуй разницу скорости (эффект raise+scale).

rapid
td="rapid" (0.1s)
fast
td="fast" (0.325s)
medium
td="medium" (0.55s)
slow
td="slow" (0.775s)

Состояние — hover / active / focus

Трансформации, фильтры, тени по состоянию. Плавность даёт td.

raise

raise — подъём

raise="hover active" td="fast"

translateY вверх при hover, ещё выше при active. Вары --raise-hover/-active.

scale

scale — масштаб

scale="hover active"

hover → 1.05, active → 0.95 (нажатие). --scale-hover/-active.

rotate — поворот

rotate="hover" · style --rotate-hover

Поворот при hover (по умолч. 3°). Здесь угол увеличен: --rotate-hover: 20deg.

skew

skew — наклон

skew="hover active"

skewX при hover/active. --skew-hover/-active.

blur

blur — размытие

blur="hover" td="medium"

Изначально размыт, при hover — резкий. --blur-amount.

gray

grayscale — обесцвечивание

grayscale="hover"

Ч/б по умолчанию, цвет при hover (зеркало blur). --grayscale-amount.

shadow

shadow — тень

shadow="hover active focus"

box-shadow при hover, плоская при нажатии. Работает и по focus.

glow

glow — свечение

glow="hover" · --glow-color/-size

Цветное неоновое свечение (в отличие от серой shadow).

Псевдоэлементы и текст

sheen

sheen — блик

sheen="hover" (::before, overflow hidden)

Белая полоса проходит по элементу. Есть sheen="" infinite.

ripple — волна

ripple="hover" (::before, без авто) · есть active

Круговая волна от центра. На hover — по наведению; на active нужно зажимать. --ripple-color.

из центра слева направо справа налево

underline — подчёркивание

underline="hover | left hover | right hover"

Анимированная линия через ::after. --underline-color/-width/-offset.

печатает текст…

typing-text — печатная машинка

typing-text="" · steps/chars = кол-во символов

Текст появляется посимвольно с курсором. Кнопка «проиграть» перезапустит.

SHIMMER

shimmer-text — мерцание текста

shimmer-text="" infinite (color → transparent)

Градиентный блик поверх текста (background-clip: text). Нужен фон для читаемости.

Комбинации — как в реальном проекте

карточка

Карточка услуги

sheen="hover" shadow="hover" raise="hover" td="medium"

Блик + тень + подъём одним наведением.

Интерактивная кнопка

scale + shadow + ripple="hover" + td="fast"

Масштаб/тень по состоянию, волна при наведении.

decor

Декоративный элемент

float="" pulse="" infinite td="slow"

Парит и пульсирует одновременно (два атрибута).

Свечение по focus

glow="hover focus" scale="hover"

Наведи мышью или перейди Tab — свечение и на фокусе.

infinite — атрибут-флаг: делает анимацию элемента бесконечной (--iteration-count: infinite). td="none" отключает переход. Все анимации автоматически глушатся при системном prefers-reduced-motion: reduce.
