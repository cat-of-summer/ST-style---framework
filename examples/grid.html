Grid-система — grid и col

12-колоночная CSS Grid через атрибуты. grid на контейнере, col на детях (ширина в колонках + позиция начала). Всё адаптивно, mobile-first. Тяни ширину окна — раскладка перестраивается по брейкпоинтам.

3 · Responsive span — col="12 md-6 lg-4"

Mobile-first: перечисляешь значения через пробел, больший брейкпоинт выигрывает.

4 · Позиция начала — start

col="4 start-1"
col="4 start-9"
col="6 start-4" (по центру)
col="3 lg-start-10"
<div col="start-2">      <!-- начать со 2-й колонки -->
<div col="4 start-2">    <!-- span 4, начало со 2-й -->
<div col="6 start-4">    <!-- span 6 по центру 12-сетки -->
<div col="4 lg-start-4"> <!-- start-4 только на lg+ -->

Значения start-1 … start-12 для каждого брейкпоинта. Управляет grid-column-start — можно оставлять «пустые» колонки.

<div grid="12 gap-x-5"> <!-- только по горизонтали 20px -->

gap-* — общий промежуток, gap-x-* / gap-y-* переопределяют одну ось. Значения 0–15 (× 4px). Тот же gap работает и на flex — см. display.

6 · Реальный паттерн: карточки

Секция с карточками: 1 в ряд на мобиле, 4 на десктопе. Тяни ширину.

Карточка 4

col="12 md-6 lg-3"

<section mt="6">
    <div container>
        <div class="cards" grid="gap-4">
            <article col="12 lg-3" sheen="hover" shadow="hover focus" tabindex="0">
                <!-- содержимое карточки -->
            </article>
        </div>
    </div>
</section>

Канонический скелет секции: section › container › grid › col. Чтобы отступы карточек «дышали» с шириной секции — оберни в container="query" (см. container-query).
