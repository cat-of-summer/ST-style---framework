Класс .editor

CSS Reset фреймворка обнуляет отступы, маркеры списков, стили ссылок — чтобы вёрстка начиналась «с чистого листа». Но HTML из CMS/WYSIWYG-редактора это ломает. .editor локально восстанавливает нормальную типографику через * { all: revert } и точечные правила.

Оборачивай в .editor любой контент, сгенерированный редактором (тело статьи, описание товара). Внутри — обычные теги без атрибутов фреймворка.

Внутри .editor сначала * { all: revert } возвращает детям браузерные дефолты, затем правила класса задают отбивки, списки, таблицы. Размеры h1–h6 берутся из токенов --fs-*, поэтому масштабируются вместе с ramp.
